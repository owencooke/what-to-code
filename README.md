## Installation

### Prerequisites

- Node.js
- NPM
- NVM

Install Yarn for dependency management

```bash
npm install --global yarn
```

Install all Node modules

```bash
yarn install
```

See `.env.dev` for the necessary environment variable template.

### Running

```bash
yarn dev
```
